# Notion2WeChat

## Setup

1. Install dependencies:

```bash
npm install
```

2. Build extension:

```bash
npm run build
```

3. Load the extension in Chrome:
- Open `chrome://extensions`
- Enable "Developer mode"
- Click "Load unpacked"
- Select this folder

## Usage

- Open a Notion page at `https://www.notion.so/*`
- Click the extension button to open the right-side drawer
- Preview the iOS-optimized WeChat HTML
- Click `Copy All` or select text in the preview and click `Copy Selection`
